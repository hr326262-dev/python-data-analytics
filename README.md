# Python Data Analytics Project

This repository contains a complete starter project for Python data analytics, including:

- A sample sales dataset
- Reusable analytics functions
- A command-line analysis script
- A Streamlit dashboard for interactive exploration
- Automated tests

## Quick start

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the script:
   ```bash
   python scripts/run_analysis.py
   ```
4. Launch the dashboard:
   ```bash
   streamlit run app.py
   ```

## Project structure

- data/ - sample CSV data
- src/data_analytics/ - analytics package
- scripts/ - command-line entry points
- tests/ - automated tests
- output/ - generated charts
